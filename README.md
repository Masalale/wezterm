# wezterm

A WSL terminal config built for Fedora. No tabs. Just panes.

![screenshot](screenshot.png)

---

## Keybindings

All actions are single-chord. No leader key. No multi-step dances.

![bindings](bindings.png)

---

> *This is what happens when a terminal config is built for one machine and one workflow.*

```bash
git clone git@github.com:Masalale/wezterm.git ~/.config/wezterm
```

On Windows: symlink `.wezterm.lua` to `%USERPROFILE%\.wezterm.lua`.

Restart WezTerm.
